This Minisite is dedicated to showing off the exercises and projects I have done using Javascript and JQuery. The menu above contains links to everything. Below is the detail of what the objectives of each exercise/project were:

  • Exercise 1: Create a basic webpage that will link to all individual items of work and describe them. Hence, this page!
    1. Files used: index.html, /css/styles.css, /common/navbar.html, /common/footer.html, /scripts/exercise1.js
    2. Create an alert upon entering the webpage to display some text.
  • Exercise 2: Use built-in functions, data, variables, and arrays. Prompt the user to enter their favorite number and website. Use the number to generate a random quote to display. List 3 of your own favorite websites and then replace the last list item with the users input to redisplay the altered list.
    1. Files used: /static/exercise2.html, /css/styles.css, /common/navbar.html, /common/footer.html, /scripts/exercise2.js
    2. Create an array with 5 quotes of the day.
    3. Prompt the user to enter a number.
    4. Divide the number by 5 and get the remainder (modulus).
    5. Use the remainder as the array index and write that quote to the page.
    6. Create an array of 3 favorite website urls.
    7. Loop through the initial url list and construct an unordered list to write them out on the web page as working links.
    8. Prompt the user to enter their favorite website and store the input in a variable.
    9. Use a loop to take the users input, replace the last list item in your initial url list with it, make it a working link too, and redisplay the updated list.
  • Exercise 3: Use conditional statements, loops, and functions. Create a quiz of 3 questions that give 3 chances to answer each correctly. Afterward, reveal the score earned per question and give the answer. Then give the total score the user earned.
    1. Files used: /static/exercise3.html, /css/styles.css, /common/navbar.html, /common/footer.html, /scripts/exercise3.js
    2. Create an array of at least 3 trivia questions.
    3. Create a second array that contains the answers to the trivia questions in the same position as the questions.
    4. Initialize a "score" variable to 0.
    5. Call the "quiz" function passing the loop counter as a parameter.
    6. Initialize a "guesses" variable to 3.
    7. Create a while loop to run as long as "guesses" > 0.
    8. Prompt the user to answer the trivia question using the counter parameter as the array index.
    9. Compare the answer entered by the user to the corresponding answer in the answers array.
    10. Write out the return of the quiz function to show the points earned per question and the answer.
    11. Write out the total score earned.
  • Exercise 4: Use strings, numbers, and date & time. Create a sale site. It will determine what month it currently is along with a specific monthly special and display both. It will also give tips for the current season and have corresponding background pictures and color schemes.
    1. Files used: /static/exercise4.html, /css/styles.css, /css/exercise4.css, /common/navbar.html, /common/footer.html, /scripts/exercise4.js
    2. Create a "months" array for the year.
    3. Create a "tips" array for 4 seasons with 3 tips each.
    4. Create a "specials" array for the year.
    5. Create a function that will popluate data based on the current date.
    6. Change the HTML elements to the appropriate season and special.
  • Exercise 5: Use forms that will validate data. Create an order form that users will fill out to place an order of products. Display error messages if required fields are not filled out. Note: The form won't fully submit at this stage because the product and shipping calculations are in another exercise.
    1. Files used: /static/exercise5.html, /css/styles.css, /css/exercise4.css, /common/navbar.html, /common/footer.html, /scripts/exercise5.js
    2. Create an order form that collects personal information, shipping address, and item order.
    3. Prevent submission of order and display error messages if required fields are not filled out.
